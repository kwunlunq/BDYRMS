程式概述:
  • 將滑鼠滑到圖片上 觸發JS的mouseover 取得圖片上資料回傳給後端(book.java)
  • 後端處裡資料取得相關BOOK的資訊 在return給前端對應的資料
  • js 接收到資料 並將結果顯示於前端畫面上
檔案清單:
  • src/book.java
  • WEB-INF/dwr.xml
  • dwrDemo/getBooks.html
JS程式碼: 
window.onload = function() {
    OPT.getOptions(populate);  
};
 
function populate(list){
	dwr.util.removeAllOptions("opts");
    dwr.util.addOptions("opts", list);
}
